# Atri Labs website

Company website built with Atri framework

## Deployed app
https://atrilabs.com/

## How to use this app?

### Step 1: Setup

```shell
https://github.com/Atri-Apps/atri-landing.git

cd atri-landing

pipenv install
```

### Step 2: Start the editor

```shell
pipenv shell

atri start
```
